# vue 与 spring mvc 联动
step1:
> [创建项目](http://start.spring.io)

step2:
> 创建vue
`vue create vue-demo`
`cd vue-demo`

step3:
> 添加路由
`vue add router`

step4:
> 编写前端代码

step5:
> 编译vue
`npm run build`

step6:
> 复制[编译的文件](./vue-demo/dist)到[spring静态文件目录](./src/main/resources/static)

step7:
> 运行
